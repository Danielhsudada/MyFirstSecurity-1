### 範例程式1: PHP101_1.php
>* 直接將 PHP 程式嵌入 HTML 網頁
```
<!doctype html>
<html>
  <head>
    <meta charset="utf-8">
    <title>天王龍的第一個PHP程式</title>
  </head>
  <body>
    <?php
      echo("Hello Dragon!");
      phpinfo();
    ?>
  </body>
</html>
```

### 範例程式: PHP101_.php
>* 將 PHP 程式放在外部檔案 然後再include近來

showme.inc
```
<?php
    echo("Hello Dragon!");
    phpinfo();
?>
```
```
<!doctype html> 
<html>
  <head>
    <meta charset="utf-8">
    <title>天王龍的第一個PHP程式</title>
  </head>
  <body>
    <?php
      include_once("showme.inc");
    ?>
  </body>
</html>
```
