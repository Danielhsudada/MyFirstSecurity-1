安裝pip ==> sudo easy_install pip

安裝virtualenv ==> pip install virtualenv

啟動openai-gym虛擬環境 ==> source openai-gym/bin/activate

安裝gym ==> pip install -U gym
      
         或sudo -H pip install -U gym[all]
           sudo chmod -R o+rw ~/openai-gym
